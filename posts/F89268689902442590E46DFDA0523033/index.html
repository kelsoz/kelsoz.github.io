markdown-文字写作利器

Wednesday, August 6, 2014 字数 607

markdown是一种简单的标记语言，对于大部分文字工作者来说，office 套件对于你们来说都太过于多余了，你们就只需要一个打字的地方加上简单的格式功能就好了嘛， office 对于你们来说又大又贵又臃肿啦。markdown 是一种简单易用而且通用的标记格式，这意味着你可以让你的文档通用于 txt 文本文件，doc 文档，html 文档，ppt 文档…… 还有一个很有用的地方在于，有一些微信运营的同学都知道微信的在线消息编辑器有多狗屎，而且微信还不提供 api 来编写离线编辑器（腾讯狗屎～），那么用 markdown 来写一段优雅的群发消息就很有用了。而且这个东西配合 everonte 团队版简直是个神器，所有你们想要的团队协作与文档共享都有现成的方案。

说了这么多，markdown 具体怎么用呢？首先你要找一个支持 markdown 的编辑器，我个人最爱的是 dayone（有 mac 版本和 ios 版本。收费，但是限免的时候收了一个啦啦啦），当然如果你习惯用 evernote 做笔记的话有一个叫个马克飞象的免费编辑器，提供在线版和 chrome 的插件版本，当然还有很多第三方的版本请自行搜索(国内的话推荐作业部落)。 具体怎么用：

  • 如果我要让一段文成为标题只要在它的前面加两个#号。
  • 如果要让几行文成为一个列表只要在它的前面加一个*号，正如现在这个列表。
  • 如果我要加粗一段文字只要再它首尾加两个*号。
  • 如果要我要加一条细线只要在空行里输入三个以上的*号。
  • 如果我要加一段引用只要在本段前面加一个>号。

其中的畅快也只有亲自去试一下才知道其中的妙处……
